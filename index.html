排序演算法視覺化專題

本網站展示五種常見排序演算法的互動式視覺化過程，
包含 Bubble Sort、Selection Sort、Insertion Sort、Merge Sort 與 Quick Sort。

Bubble Sort

透過反覆比較相鄰元素，將較大的值一步步推到右側。 是最直觀、最容易理解的基礎排序法之一。

Selection Sort

每一輪從未排序區中找出最小值， 再與前方位置交換，逐步擴大已排序區。

Insertion Sort

將元素逐一插入前面已排序好的區間中， 適合觀察「插入」與「位移」的排序思維。

Merge Sort

採用分治法，先把資料拆分成小區段， 再依序合併回來，是經典的 O(n log n) 排序法。

進入頁面

Quick Sort

選擇 pivot 作為基準，經過 partition 把資料分成左右兩邊， 再遞迴排序，是效率很高的經典演算法。
