Quick Sort Arena

這版是快速排序。它的重點不是一輪一輪找最大值，也不是拆成兩半再合併， 而是先選一個 pivot（基準值），把比它小的放左邊、比它大的放右邊， 完成一次 partition 後，pivot 會落到正確位置，接著對左右兩邊遞迴做同樣的事。

複雜度摘要

情況 時間 說明
Best Case O(n log n) 每次 partition 都能切出相對平均的左右兩邊。
Average Case O(n log n) 實務上通常很快，常被視為高效率通用排序。
Worst Case O(n²) 若 pivot 每次都選得很差，例如幾乎切不開，就會退化。
Space O(log n) 主要來自遞迴呼叫堆疊。
Stability 通常不穩定 交換過程可能改變相同元素相對順序。
這份程式採用 Lomuto partition 的視覺化思路，事件拆成 partition-start / scan / move-left / pivot-place / fixed / done， 讓 quick sort 的 pivot 與 partition 過程比較好看懂。
