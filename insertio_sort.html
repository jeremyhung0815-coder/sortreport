Insertion Sort Studio

這版是插入排序的互動式視覺化。核心邏輯不是「一路把最大值往右推」，而是每次挑出一個 key， 往左掃描已排序區，把比它大的元素往右移，最後把 key 插進正確位置。 所以你會看到這份程式的動畫重點改成：選 key、向左比較、右移元素、插回空位。

複雜度摘要

情況 時間 說明
Best Case O(n) 陣列原本就接近有序，只需少量比較。
Average Case O(n²) 一般情況下，常需要反覆向左掃描與移動。
Worst Case O(n²) 完全逆序時，每個新元素都要插到最前面。
Space O(1) 原地排序，只額外使用少量暫存變數。
Stability 穩定 相等元素的相對順序可維持不變。
這份程式刻意把動畫事件拆成 pick / compare / shift / insert / round-end / done， 讓插入排序的動態特徵更清楚，也和你前一個 bubble sort 版本的互動節奏區隔開來。
